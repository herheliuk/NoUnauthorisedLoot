Plugin requires no configuration and works out of the box.

Prevents players from accessing any foreign objects, even with no TC, including boxes, furnaces, etc.

Useful for PvE servers, as players can still access others’ belongings by default.

Other useful plugins: [Anti Ladder and Twig](https://umod.org/plugins/anti-ladder-and-twig), [No Player Loot](https://umod.org/plugins/no-player-loot) and [No Unauthorised Damage](https://umod.org/plugins/no-unauthorised-damage).
